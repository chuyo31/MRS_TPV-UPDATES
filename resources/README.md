# Recursos

Esta carpeta contiene recursos de la aplicación.

## Archivos Requeridos

- `icon.ico` - Icono de la aplicación para Windows (formato ICO, múltiples tamaños recomendados)

## Instrucciones

1. Crea o coloca un archivo `icon.ico` en esta carpeta
2. El icono se usará en el instalador y en la barra de tareas de Windows
